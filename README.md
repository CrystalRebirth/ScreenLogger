- 🤖 Discord webhook integration
- 🏗️ Easy EXE conversion
- 🔒 Configurable capture settings
- 📈 System monitoring capabilities

## 🛠️ Setup
1. Run the automated setup:
```bash
setup.bat
```

2. Configure your Discord webhook in `config.py`

3. Build the EXE:
```bash
python main.py
```
